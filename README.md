# Describe Table  
  
## A PHP PDO Script to print Column Names and Type  
  
For those times when you need a simple list of column info.  
  
Sample output:

id - int(11) unsigned  
fullname - varchar(64)  
email - varchar(96)  
phone - varchar(22)  
message - text  
created - timestamp  
updated - datetime  
deleted - tinyint(1)
